Σύγκριση
Lucide ✓ Προτείνω
— CDN ready, βάζεις με ένα script tag
— Consistent 2px stroke παντού
— Τέλεια για dark themes
— Tree-shakeable αν πας σε bundler
— 1000+ icons, όλα που χρειάζεσαι υπάρχουν
Phosphor
— Πιο expressive, 6 weights
— Δεν έχει εύκολο CDN για browser
— Χρειάζεται απόφαση για weight
— Καλύτερο για React/bundler projects
— Overkill για static site
